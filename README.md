# NextJS Ignite Shop
__Ignite 2022 By Rocketseat__

## Tecnologias usadas
- NextJS
- Typescript
- Stripe Pagamentos
- Axios
- Stitches CSS-in-JS
- Keen Slider

### Instalações neste Projeto
Typescript
```
npm i typescript @types/react @types/node -D
```
Stitches
```
npm i @stitches/react
```
Keen Slider
```
npm i keen-slider
```
Stripe
```
npm i stripe
```
Axios
```
npm i axios
```
